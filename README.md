# JavaFX Particle
Experimento de implementação de explosão fazendo uso de partículas com JavaFX.

## Como funciona
Faça cliques com o mouse na janela e a cada clique uma explosão acontece.

## Pré-requisito
- JDK 25 ou superior.

## Executar o projeto
Abra o terminal na pasta raiz do projeto e use o comando apropriado para o seu Sistema Operacional:

| Sistema Operacional | Comando                     |
| :--- |:----------------------------|
| **Linux/macOS** | `./mvnw clean javafx:run`   |
| **Windows** | `mvnw.cmd clean javafx:run` |
